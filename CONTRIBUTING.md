# Contributing to Agentic OS 2026

First off, thank you for considering contributing to this repository! It's people like you that make the AI Agent ecosystem such a great community.

## 🤝 How Can I Contribute?

### 1. Reporting Bugs
If you notice a typo, broken link, or factual error in the content, please open an Issue. Use the `Bug Report` template.

### 2. Suggesting New Content
The AI Agent landscape is evolving rapidly. If you think a new framework, tool, or platform should be added to `AWESOME_AGENTS.md` or a new chapter in the book:
- Open a `Feature Request` issue to discuss the proposed addition.
- Once approved, you can submit a Pull Request.

### 3. Submitting a Pull Request (PR)
1. Fork the repository and create your branch from `main`.
2. If you've added code or text, ensure it follows the existing formatting.
3. Update the `README.md` or `AWESOME_AGENTS.md` if necessary.
4. Issue that pull request!

## 📝 Code of Conduct
Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
