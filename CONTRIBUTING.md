# Contributing to 2026 AI Agent 史诗全书

We love your input! We want to make contributing to this project as easy and transparent as possible.

## Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added content, update the VitePress navigation/sidebar in `docs/.vitepress/config.js` if necessary.
3. Ensure your markdown is properly formatted.
4. Issue that pull request!

## Content Guidelines
- **Fact-Based**: Please ensure all updates regarding AI Agents are based on confirmed data from 2026.
- **Image Hosting**: Store images in `docs/assets` and reference them relatively (`./assets/image.png`). Do not use unreliable CDNs.

## Code of Conduct
Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
