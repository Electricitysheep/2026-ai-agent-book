## 🏗️ Frameworks & Runtimes (框架与运行时)
*The core libraries for building autonomous agents.*

- [AutoClaw](https://github.com/THUDM/AutoClaw) - The next-generation action-space expansion framework by Zhipu AI.
- [LangChain](https://github.com/langchain-ai/langchain) - The classic framework for developing applications powered by language models.
- [LlamaIndex](https://github.com/run-llama/llama_index) - Data framework for connecting custom data sources to large language models.
- [AutoGen](https://github.com/microsoft/autogen) - Microsoft's multi-agent conversation framework.
- [MetaGPT](https://github.com/geekan/MetaGPT) - The Multi-Agent Framework acting as a software company.

## 🔌 Protocols & Standards (协议与标准)
*Protocols enabling agents to communicate with tools and each other.*

- [Model Context Protocol (MCP)](https://github.com/modelcontextprotocol) - Anthropic's open standard that connects AI models to data sources and tools seamlessly.
- [Agent Protocol](https://agentprotocol.ai/) - A single common interface for communicating with agents.

## ☁️ Platforms (云端平台)
*No-code and low-code platforms for deploying agents.*

- [Coze](https://www.coze.com/) - ByteDance's ultimate bot-building platform.
- [Dify](https://github.com/langgenius/dify) - An open-source LLM app development platform.
- [Hailuo (MiniMax)](https://hailuoai.com/) - Deep multimodal integration platform by MiniMax.

## 💻 Hardware & OS (硬件与操作系统)
*Native operating system integrations and agentic devices.*

- [CodeBuddy](https://github.com/Tencent/CodeBuddy) - Tencent's PC desktop native assistant deep integration.
- [Comate](https://comate.baidu.com/) - Baidu's deeply integrated IDE and OS-level assistant.
